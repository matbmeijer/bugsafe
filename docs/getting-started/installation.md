# Installation

## Requirements

- Python 3.10 or higher
- pip or uv package manager

## Install from PyPI

=== "pip"

    ```bash
    pip install bugsafe
    ```

=== "uv"

    ```bash
    uv add bugsafe
    ```

## Verify Installation

```bash
bugsafe --version
```
